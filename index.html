Gazetteer

A project using leaflet.js containing a world map, allowing the user to select different countries. The selected country would then become highlighted with different informative markers dotted around the country showing the location and name of the capital city, location and size or recent earthquakes, and current weather conditions. Buttons were also placed to the side to display different popups of information about each country upon the respective buttons being clicked. This was acomplished using third party RESTful API's to gather upto date data on the selected country once a country had been selected and using the returned JSON data to overwrite the old data.
